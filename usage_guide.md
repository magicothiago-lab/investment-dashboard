## ðŸ“Š Understanding Your Dashboard

### KPI Cards (Top Section)

**Total Invested**
- Your total capital across all platforms
- Shows change from previous month if historical data exists

**Total Received**
- Total returns you've received
- Shows change from previous month

**Portfolio Yield**
- Percentage return: Received Ã· Invested
- Shows if your portfolio is performing well

**Total Profit/Loss**
- Simple math: Received - Invested
- Green = profit, Red = loss
- Also shows ROI (Return on Investment)

### Charts

**Investment Distribution (Donut Chart)**
- See what percentage each platform represents
- Helps identify if you're too concentrated in one platform

**Invested by Platform (Bar Chart)**
- Visual comparison of your capital allocation
- Quickly spot your largest positions

**Returns by Platform (Bar Chart)**
- See which platforms are returning the most
- Compare returns across your portfolio

**Portfolio Performance Over Time (Line Chart)**
- Only appears if you upload historical data
- Shows how your portfolio has grown month-over-month
- Three lines: Invested, Received, and Profit/Loss

### Performance Metrics Table

Shows detailed metrics for each platform:
- **P&L**: Profit or Loss in euros
- **ROI**: Return on Investment percentage
- **Yield**: Current return rate
- **Share**: What % of your portfolio this platform represents

You can:
- Search for specific platforms
- Sort by any column (click header)
- Change rows per page (10, 25, 50, 100)

## ðŸ”§ Troubleshooting

### CSV Not Loading
- Check column names: `Platform`, `Invested (â‚¬)`, `Received`
- Ensure no empty rows at top
- Verify number formats
- Try the demo CSV first

### Historical Chart Not Showing
- Month column must be in YYYY-MM format
- Examples: `2024-12`, `2024-11`
- Not: `December`, `12/2024`, `Dec-24`

### Notifications Not Working
- Click the bell icon to enable
- Check browser/phone settings
- Ensure app has notification permission
- Notifications only work after goal is set

### Widget Not Updating (Android)
- Open the app to force update
- Check if app is in battery optimization (disable it)
- Remove and re-add widget

### PDF Export Issues
- Try a different browser (Chrome recommended)
- Check if pop-ups are blocked
- Ensure enough storage space

### App Not Installing
- **iOS**: Must use Safari browser
- **Android**: Must use Chrome browser
- Try incognito/private mode
- Clear browser cache
